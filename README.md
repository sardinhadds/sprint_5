# Sprint5
## Anúncio online de carros
Este app tem como finalidade apresentar uma análise exploratória de dados de venda de veículos obtidos em https://practicum-content.s3.us-west-1.amazonaws.com/new-markets/Data_NEW_4_sprint/vehicles.csv.
Você encontrará dois botões. Um que apresenta um histograma dos dados e outro botão que apresenta um gráfico de dispersão.
Aperte em um dos botões e o gráfico correspondente será exibido.
